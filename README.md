# Galaxy_Potentials_and_Orbits_Integration
Galaxy_Potentials_and_Orbits_Integration simulation codes for the Física Galáctica class in Máster de Astrofísica of Universidad  de La Laguna.

In this GitHub repository we attach the simulation codes for the orbits of the Helmi streams members under the Milky Way potential and under the Milky Way and M31 composite potential. It is necessary to install the numpy, matplotlib, astropy and gala Python packages for the codes to be executed.
